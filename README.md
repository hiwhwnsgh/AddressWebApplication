## 주요 기능
### 🌟 주소록 리스트 출력
- Mysql DB에 저장 되어있는 데이터를 가져와 리스토로 출력하여 제공
### 🌟 주소록 저장,수정 및 삭제 가능
- 주소록 저장,수정,삭제 기능 제공
### 🌟 프로필 접기&펼치기 기능
- 주소록의 프로필 하나를 클릭하면 세부정보가 간단히 출력
